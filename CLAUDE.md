# ThinkLight — notes for Claude Code

- After committing, push to origin by default (no need to ask).
- The CLI (`src/thinklight`) is installed to `~/.local/bin` by `install.sh`;
  hooks run the installed copy, so reinstall after editing it when testing.
- Keep README.md (Chinese, the GitHub landing page) and README.en.md
  (English) in sync when documenting behavior.
